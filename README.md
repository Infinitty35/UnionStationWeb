# Union Station South

Static website for Union Station South HOA. Single self-contained `index.html` — no build step.

## Deploy (Cloudflare Pages)

Connected to this repo. Build settings:

- **Framework preset:** None
- **Build command:** *(empty)*
- **Build output directory:** `/`

Every push to `main` auto-deploys.

## Local preview

Just open `index.html` in a browser, or serve the folder:

```
python3 -m http.server 8000
```
